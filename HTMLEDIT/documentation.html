LA DOCUMENTATION DE HTMLEDIT

1- fonctionnement principale:

Le but de ce logiciel est de fournir du bon soft afin de pouvoir tapersereinement du code HTML et de le tester.

2- Les 4 grands bouttons :

Ces 4 boutons servent à : ouvrir un fichier(le fichier code.HTML), sauvegarder(dans code.HTML), à lancer le fichier code.HTML(après sauvegarder) et à quitter, chacun dans une couleur.
